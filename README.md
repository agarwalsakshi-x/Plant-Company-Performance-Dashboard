# Plant Company Performance Dashboard (Power BI)

## Overview
This project presents an interactive **Power BI performance dashboard** analyzing **Sales, Gross Profit, and Quantity** across time, countries, and product categories.
The dashboard converts raw transactional data into **decision-ready business insights** that help stakeholders monitor performance, identify declines, and evaluate growth opportunities.

## Business Objective
Organizations require clear visibility into:

- Year-to-date (**YTD**) performance
- Comparison with **previous year-to-date (PYTD)**
- Country-level underperformance
- Monthly performance trends by product type
- Profitability distribution across accounts

This dashboard enables **data-driven performance monitoring and strategic decision-making**.

## Dashboard Pages & Analysis

### 1. Gross Profit Performance
- KPI cards for **YTD, PYTD, YTD vs PYTD, and GP%**
- **Bottom 10 countries** by YTD performance using treemap visualization
- **Monthly waterfall chart** showing increases and decreases vs PYTD
- **Trend analysis by product type**
- **Account probability segmentation** using GP% vs Value scatter plot

### 2. Sales Performance
- Year-to-date **Sales KPIs** with PYTD comparison
- Country-level underperformance identification
- Monthly **sales trend and variance analysis**
- Scatter visualization linking **profitability and sales value**

### 3. Quantity Performance
- KPI monitoring for **quantity movement vs previous year**
- Monthly increase/decrease contribution analysis
- Product-type quantity trends
- Profitability vs quantity relationship across accounts

## Key Features

- Interactive **filters and slicers** for dynamic exploration  
- **DAX-based KPI calculations** for YTD, PYTD, variance, and GP%  
- Multi-page **executive performance reporting layout**  
- Combination of **treemap, waterfall, trend, and scatter visuals**  
- Designed for **business stakeholders and leadership monitoring**

## Business Value
This dashboard demonstrates how operational data can be transformed into:

- Performance monitoring dashboards  
- Profitability and decline detection  
- Country and product-level insights  
- Executive-ready reporting for strategic planning

## Tools & Technologies
- **Power BI**
- **DAX**
- **Data Modeling**
- **Interactive Data Visualization**
